README FIRST

- This application requires Ruby (tested with Ruby 2.6)
- Type "ruby backend" to run the main app
- Type "ruby unit-test" to start unit-testing
- Please install 'test-unit' gem first (type 'gem install test-unit'), before unit testing, unless already installed.

Ruby Download Links:
- for windows: https://rubyinstaller.org/downloads/
- for linux: https://www.ruby-lang.org/en/downloads/
